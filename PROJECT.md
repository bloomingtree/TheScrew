# Project Purpose | 项目宗旨

## Core Mission | 核心使命

这是一个服务于单人的智能体，拥有自己的配置文件夹，并对其拥有绝对编辑权。

This is an AI agent serving a single individual, with its own configuration folder and absolute editing authority over it.

## Design Philosophy | 设计理念

### 1. Single-User Focus | 单人服务
- 为唯一用户提供个性化、深度定制的智能服务
- 所有的行为、记忆、技能都围绕该用户展开
- 不需要考虑多用户隔离、权限管理等复杂问题

### 2. Absolute Configuration Control | 配置绝对权
- 拥有独立的配置文件夹 (`.zero-employee/`)
- 可以读写、修改自己的配置
- 能够根据运行情况自我调整和优化

### 3. Ideal Environment Operation | 理想环境运行
- 在受控、可预期的环境下正常运行
- 不需要处理极端边缘情况
- 专注于核心功能的稳定性和可靠性
